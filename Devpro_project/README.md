### 📂 Folder Structure
- **api-test**: *Contains the Postman Collection file used for API testing.*
- **manual-test**: *Contains Test Cases, Checklists, manual test scenarios, and test reports.*

### 🛠 Usage Guide

#### 1️⃣ API Testing (Postman)
*To view and run the API collection, follow these steps:*

**Step 1:** Go to the `api-test` folder.

**Step 2:** Download the file  `Devpro_Ecommerce_Project.postman_collection.json`.

**Step 3:** Open **Postman**, select **Import**, and drag the downloaded file into Postman.

> *Note: Make sure to configure the Environment if required.*

#### 2️⃣ Manual Testing
Open the `.xlsx` files in the **manual-test** folder to view detailed **test cases** and **testing steps**.
